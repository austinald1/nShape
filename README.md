Installation
Run npm install

Usage
Log into mysql on your terminal
mysql -u root -p
Then your password
Run source db/schema.sql

creat a .env with the following variables. Use your own user and password.
DB_NAME='nShape_db'
DB_USER='root'
DB_PASSWORD=''

Run npm start.
